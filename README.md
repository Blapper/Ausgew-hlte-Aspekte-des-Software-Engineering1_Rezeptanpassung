# Ausgew-hlte-Aspekte-des-Software-Engineering1_Rezeptanpassung
Ziel des Projektes ist es die Abgabe des Assignment für die VL des Modeles AAdSE1. Das Projekt beschäftigt sich mit der optimierung der Mengen für Lagerküchen für Kinder und Jugendlagern. Es wird protokolliert wie viele von welcher Mahlzeit übrig geblieben ist. Hierdurch kann mathematisch der zukünftige Verbrauch abgeschätzt werden.
